# UI Layouts

## 17.1 Dashboard
- Workspace selector
- Project list
- Create project button

## 17.2 Project Page
- Project name, client
- Sources list (type icon, name, date, snippet, quality badge)
- Add source button (modal: Paste Notes / Paste Email / Upload)
- Generate Story Pack button
- Packs list

## 17.3 Pack Editor (Three-Panel)
**LEFT (280px, collapsible)**: Nav tree - Summary, Non-Goals, Stories, Assumptions, Decisions, Risks, Open Questions
**CENTRE (fluid)**: Story card - persona, want, soThat. ACs in GWT format. Evidence count badge. Unsupported badge.
**RIGHT (320px, collapsible)**: Tabs - QA Flags, Evidence, Changes
**TOP BAR**: Pack name, version badge, breadcrumb

## 17.4 Review Page (/review/[token])
- Read-only pack view
- Comment threads on stories/ACs
- Responsive 768px+
